Tymbark API to zbiór 523 haseł z napojów marki Tymbark

Przykłady użycia:

GET: 
https://tymbark-api.k0pa.repl.co

Python:

import requests
haslo = requests.get("https://tymbark-api.k0pa.repl.co")
haslo = haslo.json()
haselko = haslo['haslo']
numer = haslo['numer']
print(haselko)
print(numer)
